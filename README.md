# Energy Grid Demand Analytics Dashboard

This project analyzes electricity demand patterns using an interactive Tableau dashboard.

## Project Overview

The dashboard explores electricity consumption trends across time to identify peak demand periods and usage patterns.

## Key Insights

- Identifies peak electricity demand hours
- Shows daily and hourly consumption trends
- Highlights variations in energy usage

## Tools Used

- Tableau Public
- Data Visualization
- Time Series Analysis
